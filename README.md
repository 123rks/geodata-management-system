## Description

GeoData Management System (NestJS)

## Installation

Set ENV variables in .env file
```bash
$ npm install
```
Run migrations to create tables in the database
```bash
$ npm run typeorm:run-migrations
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
$ npm run test
```
